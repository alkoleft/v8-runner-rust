# AgentListenAddress

## Назначение

Позволяет указать IP`-адрес`, который будет прослушиваться агентом. Если параметр не указан, то по умолчанию используется IP`-адрес` 127.0.0.1.

## Синтаксис

```text
/AgentListenAddress <Адрес>
```

## Связи

- Если параметр не указан, то по умолчанию используется IP`-адрес` 127.0.0.1.
