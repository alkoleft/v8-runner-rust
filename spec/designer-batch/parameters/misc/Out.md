# Out

## Назначение

Установка файла для вывода служебных сообщений. Если задан параметр `-NoTruncate` (через пробел), файл не очищается (не используется в тонком клиенте).

## Синтаксис

```text
/Out <имя файла> [-NoTruncate]
```

## Связи

- Если задан параметр `-NoTruncate` (через пробел), файл не очищается (не используется в тонком клиенте).

## Примечания

- Во время исполнения пакетных команд файл сообщений можно открыть для просмотра.
- В пакетном режиме запись сообщений в файл не буферизуется (сообщения записываются сразу).
- Файл формируется в кодировке UTF`-8`.
