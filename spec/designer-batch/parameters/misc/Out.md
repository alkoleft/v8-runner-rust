# Out

## Синтаксис

```text
/Out <имя файла> [-NoTruncate]
```

## Описание

— установка файла для вывода служебных сообщений. Если задан параметр **-NoTruncate** (через пробел), файл не очищается (не используется в тонком клиенте).
Во время исполнения пакетных команд файл сообщений можно открыть для просмотра. В пакетном режиме запись сообщений в файл не буферизуется (сообщения записываются сразу). Файл формируется в кодировке UTF-8.
