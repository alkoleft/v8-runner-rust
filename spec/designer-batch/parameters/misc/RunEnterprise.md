# RunEnterprise

## Назначение

Запуск "1С:Предприятия" после исполнения пакетной команды. После параметра может быть указана дополнительная командная строка. При запуске "1С:Предприятия", переданные в ней параметры будут использованы вместо параметров текущей сессии. Дополнительная командная строка обязательно должна быть в кавычках, вложенные в нее кавычки должны быть удвоены.

## Синтаксис

```text
/RunEnterprise
```
