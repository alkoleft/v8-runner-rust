# DisableStartupDialogs

## Назначение

Отключает стартовый диалог и диалоги аутентификации при запуске.

## Синтаксис

```text
/DisableStartupDialogs
```

## Связи

- Если командной строки недостаточно для выбора базы, режима запуска или авторизации, вместо диалога команда завершится ошибкой.
- Если в командной строке нет авторизации в хранилище конфигурации, конфигуратор стартует без связи с хранилищем.
- В режиме `/AgentMode` параметр игнорируется.

## Примечания

- Параметр действует ограниченное время: в режиме `1С:Предприятие` его действие заканчивается после обработки события `ПередНачаломРаботыСистемы`.
