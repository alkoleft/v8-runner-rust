# CreateDistributive

## Назначение

Создает комплект поставки или набор файлов поставки по готовому описанию комплекта.

## Синтаксис

```text
/CreateDistributive <каталог создания комплекта поставки> -File <имя файла описания комплекта поставки>
[-Option <вариант поставки>] [-MakeSetup] [-MakeFiles] [-digisign <имя файла с параметрами лицензирования>] [-WarningsAsErrors]
```

## Параметры

- `<каталог создания комплекта поставки>` — каталог, куда будет собран результат.
- `-File <имя файла описания комплекта поставки>` — описание комплекта поставки.
- `-Option <вариант поставки>` — имя варианта поставки из описания.
- `-MakeSetup` — создает комплект поставки.
- `-MakeFiles` — создает только файлы комплекта поставки.
- `-digisign <имя файла с параметрами лицензирования>` — параметры лицензирования рабочего места.
- `-WarningsAsErrors` — трактует предупреждения как ошибки.

## Связи

- `-MakeSetup` и `-MakeFiles` взаимоисключают друг друга.
- Если ни `-MakeSetup`, ни `-MakeFiles` не указаны, используется `-MakeSetup`.
- Если `-Option <вариант поставки>` не указан, используется вариант `Полный`.
- `-WarningsAsErrors` делает несоответствие подписи и метаданных фатальной ошибкой.

## Примечания

- Команда устарела и не рекомендуется для использования.
- Все каталоги в полном пути к входным и выходным файлам должны существовать заранее.
