# CreateDistributionFiles

## Назначение

Создает файлы поставки и обновления.

## Синтаксис

```text
/CreateDistributionFiles [-cffile <имя cf-файла>]
[-cfufile <имя cfu-файла> [-f <имя cf-файла>|-v <версия дистрибутива>]+]
[-digisign <имя файла с параметрами лицензирования>][-WarningsAsErrors]
```

## Параметры

- `-cffile <имя cf-файла>` — создает дистрибутив.
- `-cfufile <имя cfu-файла>` — создает файл обновления.
- `-f <имя cf-файла>` — включает дистрибутив в обновление по имени файла.
- `-v <версия дистрибутива>` — включает дистрибутив в обновление по номеру версии.
- `-digisign <имя файла с параметрами лицензирования>` — параметры лицензирования рабочего места.
- `-WarningsAsErrors` — трактует предупреждения как ошибки.

## Связи

- Группа `-f <имя cf-файла>` или `-v <версия дистрибутива>` повторяется столько раз, сколько дистрибутивов нужно включить в обновление.
- `-WarningsAsErrors` делает несоответствие подписи и метаданных фатальной ошибкой.

## Примечания

- Все каталоги в полном пути к входным и выходным файлам должны существовать заранее.
