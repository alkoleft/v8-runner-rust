# CreateTemplateListFile

## Назначение

Создает файл со списком шаблонов конфигураций.

## Синтаксис

```text
/CreateTemplateListFile <имя файла> [-TemplatesSourcePath]
```

## Параметры

- `<имя файла>` — файл результата.
- `-TemplatesSourcePath` — каталог поиска файлов шаблонов.

## Связи

- Если указан только файл без пути, он создается в целевом каталоге с этим именем.
- Если путь не указан для `-TemplatesSourcePath`, используется путь из системных настроек запуска.

## Примечания

- Все каталоги в полном пути к файлу результата должны существовать заранее.
