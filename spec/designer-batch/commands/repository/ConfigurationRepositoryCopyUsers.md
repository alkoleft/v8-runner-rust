# ConfigurationRepositoryCopyUsers

## Назначение

Копирует пользователей из другого хранилища конфигурации.

## Синтаксис

```text
/ConfigurationRepositoryCopyUsers -Path <путь> -User <Имя>
-Pwd <Пароль> [-RestoreDeletedUser][-Extension <имя расширения>]
```

## Параметры

- `-Path <путь>` — источник, из которого копируются пользователи.
- `-User <Имя>` — пользователь исходного хранилища.
- `-Pwd <Пароль>` — пароль пользователя исходного хранилища.
- `-RestoreDeletedUser` — восстанавливает удаленного пользователя, если найден пользователь с тем же именем.
- `-Extension <имя расширения>` — работает с хранилищем указанного расширения вместо основной конфигурации.

## Связи

- Удаленные пользователи не копируются, если `-RestoreDeletedUser` не указан.
- Если пользователь с таким именем уже существует, новый пользователь не добавляется.
- Если `-Extension <имя расширения>` не указан, команда работает с хранилищем основной конфигурации.
