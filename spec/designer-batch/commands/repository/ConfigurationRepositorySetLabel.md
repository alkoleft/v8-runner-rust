# ConfigurationRepositorySetLabel

## Назначение

Устанавливает метку для версии хранилища конфигурации.

## Синтаксис

```text
/ConfigurationRepositorySetLabel [-Extension <имя расширения>]
[-v <номер версии хранилища>] [-name"<название метки>"] [-comment"<текст комментария>"]
```

## Параметры

- `-Extension <имя расширения>` — работает с хранилищем указанного расширения.
- `-v <номер версии хранилища>` — версия, для которой ставится метка.
- `-name"<название метки>"` — имя метки.
- `-comment"<текст комментария>"` — комментарий к метке.

## Связи

- Если `-v <номер версии хранилища>` не указан, метка ставится на последнюю версию.
- Если указана несуществующая версия, команда завершится ошибкой.
- Для многострочного комментария параметр `-comment"<текст комментария>"` можно повторять по одному разу на строку.
