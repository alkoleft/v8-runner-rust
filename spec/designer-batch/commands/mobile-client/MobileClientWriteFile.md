# MobileClientWriteFile

## Назначение

Записывает xml`-файл` мобильного клиента с заданным именем. Возможно предварительное обновление конфигурации базы данных.

## Синтаксис

```text
/MobileClientWriteFile <имя файла>
```

## Связи

- Если в параметре командной строки требуется ввести имя файла, то следует учитывать, что при указании имени файла с полным путем все каталоги, входящие в состав пути, должны существовать.
