# UpdateCfg

## Назначение

Обновляет конфигурацию, находящуюся на поддержке, по `cf` или `cfu` файлу и настройкам объединения.

## Синтаксис

```text
/UpdateCfg <имя cf | cfu-файла> -Settings <имя файла настроек>
[-IncludeObjectsByUnresolvedRefs | -ClearUnresolvedRefs][-force][-DumpListOfTwiceChangedProperties]
```

## Параметры

- `-Settings <имя файла настроек>` — файл настроек объединения.
- `-IncludeObjectsByUnresolvedRefs` — автоматически включает в обновление объекты, на которые есть неразрешенные ссылки.
- `-ClearUnresolvedRefs` — очищает ссылки на объекты, не вошедшие в обновление.
- `-force` — продолжает обновление несмотря на предупреждения.
- `-DumpListOfTwiceChangedProperties` — выводит список дважды измененных свойств.

## Связи

- `-IncludeObjectsByUnresolvedRefs` и `-ClearUnresolvedRefs` задают альтернативное поведение для неразрешенных ссылок.
- Без `-force` объединение прерывается на предупреждениях о применении настроек, дважды измененных свойствах и удаляемых объектах со ссылками.

## Примечания

- `-force` не убирает сами предупреждения, а только не делает их блокирующими.
