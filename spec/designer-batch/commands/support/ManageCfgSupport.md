# ManageCfgSupport

## Назначение

Управляет режимом поддержки конфигурации.

## Синтаксис

```text
/ManageCfgSupport [-disableSupport] [-force]
```

## Параметры

- `-disableSupport` — снимает конфигурацию с поддержки.
- `-force` — снимает с поддержки даже если в конфигурации запрещены изменения.

## Связи

- Без `-disableSupport` команда завершится ошибкой.
- Если изменения запрещены и `-force` не указан, снятие с поддержки не выполняется.
