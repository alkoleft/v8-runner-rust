# RestoreIB

## Назначение

Загружает информационную базу из файла выгрузки.

## Синтаксис

```text
/RestoreIB <имя файла> [-JobsCount <Количество>]
```

## Параметры

- `<имя файла>` — файл выгрузки информационной базы.
- `-JobsCount <Количество>` — число фоновых заданий для загрузки.

## Связи

- Значение по умолчанию для `-JobsCount <Количество>`: `0`.
- При `-JobsCount 0` платформа автоматически выбирает число фоновых заданий по числу процессоров.
- Ускорение от `-JobsCount <Количество>` обычно есть в клиент-серверном сценарии с быстрым каналом между кластером и СУБД.

## Примечания

- Если файл информационной базы отсутствует в указанном каталоге, платформа создаст новую информационную базу.
- Все каталоги в полном пути к файлу выгрузки должны существовать заранее.
