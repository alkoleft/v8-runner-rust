# DumpDBCfgList

## Назначение

Вывод расширений конфигурации. Если ни один параметр не задан, будет выдана ошибка. Доступные параметры:.

## Синтаксис

```text
/DumpDBCfgList [-Extension <имя расширения>] [-AllExtensions]
```

## Параметры

- `-Extension <Имя расширения>` — выводит имя конфигурации`-расширения`.
- `-AllExtensions` — выводит имена всех расширений.

## Примечания

- Если ни один параметр не задан, будет выдана ошибка.
- `-Extension <Имя расширения>`: если расширение не существует, будет выдана ошибка;
