# RollbackCfg

## Назначение

Возврат к конфигурации базы данных. Доступные параметры:.

## Синтаксис

```text
/RollbackCfg [-Extension <имя расширения>]
```

## Параметры

- `-Extension <Имя расширения>` — обработка расширения с указанным именем.

## Примечания

- `-Extension <Имя расширения>`: если расширение успешно обработано, возвращает код возврата 0, в противном случае (если расширение с указанным именем не существует или в процессе работы произошли ошибки) — 1.
