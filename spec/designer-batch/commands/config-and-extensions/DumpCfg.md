# DumpCfg

## Синтаксис

```text
/DumpCfg <имя cf-файла> [-Extension <имя расширения>]
```

## Описание

### Параметры

- ** **— сохранение конфигурации или расширения конфигурации в файл. Доступные параметры:

- **Extension <Имя расширения>** — сохранение расширения с указанным именем. Если расширение успешно обработано возвращает код возврата 0, в противном случае (если расширение с указанным именем не существует или в процессе работы произошли ошибки) — 1.

**Внимание!** Если в параметре командной строки требуется ввести имя файла, то следует учитывать, что при указании имени файла с полным путем все каталоги, входящие в состав пути, должны существовать.
