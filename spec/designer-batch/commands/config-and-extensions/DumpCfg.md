# DumpCfg

## Назначение

Сохраняет конфигурацию или расширение в `cf`-файл.

## Синтаксис

```text
/DumpCfg <имя cf-файла> [-Extension <имя расширения>]
```

## Параметры

- `<имя cf-файла>` — файл результата.
- `-Extension <имя расширения>` — сохраняет только указанное расширение.

## Примечания

- `-Extension <имя расширения>` возвращает `0` при успехе и `1`, если расширение не найдено или операция завершилась ошибкой.
- Все каталоги в полном пути к `cf`-файлу должны существовать заранее.
