# LoadConfigFiles

## Назначение

Загружает отдельные свойства метаданных конфигурации из файлов: модули, макеты, картинки, права и справку.

## Синтаксис

```text
/LoadConfigFiles <каталог выгрузки> [-Module] [-Template]
[-Help] [-AllWritable] [-Picture] [-Right][-Extension <имя расширения>]
```

## Параметры

- `<каталог выгрузки>` — каталог с файлами свойств.
- `-Module` — загружает модули.
- `-Template` — загружает макеты.
- `-Help` — загружает справочную информацию.
- `-AllWritable` — загружает только свойства объектов, доступных для записи.
- `-Picture` — загружает общие картинки.
- `-Right` — загружает права.
- `-Extension <имя расширения>` — загружает свойства в указанное расширение.

## Связи

- Если `-Extension <имя расширения>` подключено к хранилищу, изменяемые объекты должны быть захвачены в хранилище.

## Примечания

- `-Extension <имя расширения>` возвращает `0` при успехе и `1`, если расширение не найдено или операция завершилась ошибкой.
