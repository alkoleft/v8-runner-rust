# LoadConfigFromFiles

## Назначение

Загружает конфигурацию или расширения из XML-файлов. Поддерживает полную, частичную и архивную загрузку.

## Синтаксис

```text
/LoadConfigFromFiles <каталог загрузки> [-Extension <имя расширения>]
[-AllExtensions][-files "<файлы>"][-partial][-listFile <файл списка>][-format <режим>] [-updateConfigDumpInfo]
[-NoCheck][-Archive <имя ZIP-архива>]
```

## Параметры

- `<каталог загрузки>` — каталог с XML-файлами конфигурации.
- `-Extension <имя расширения>` — загружает только указанное расширение.
- `-AllExtensions` — загружает все расширения из подкаталогов.
- `-files "<файлы>"` — список файлов для выборочной загрузки через запятую.
- `-partial` — загружает только выбранные файлы и отдельные внешние свойства.
- `-listFile <файл списка>` — берет список файлов из UTF-8 файла.
- `-format Hierarchical|Plain` — задает формат выборочной загрузки.
- `-updateConfigDumpInfo` — обновляет `ConfigDumpInfo.xml` после загрузки.
- `-NoCheck` — отключает проверку целостности загружаемой конфигурации.
- `-Archive <имя ZIP-архива>` — загружает конфигурацию из ZIP-архива.

## Связи

- Загрузка расширения в основную конфигурацию и загрузка основной конфигурации в расширение не поддерживаются.
- Если `-Extension <имя расширения>` связано с хранилищем, доступна только частичная загрузка и только для захваченных объектов.
- `-AllExtensions` работает только с расширениями.
- `-files` и `-listFile` взаимозаменяемы; если указаны оба, используется тот, который встретился в командной строке первым.
- `-partial` применяется к выборочной загрузке по `-files` или `-listFile`.
- По умолчанию `-format` использует `Hierarchical`.
- Для выполнения команды нужен либо `<каталог загрузки>`, либо `-Archive <имя ZIP-архива>`.

## Примечания

- При запуске в режиме агента пути в `-files` и `-listFile` должны быть относительными.
- Файл для `-listFile` должен быть в UTF-8, без пустых строк между именами.
- `-AllExtensions` создает отсутствующие расширения при необходимости.
- `-updateConfigDumpInfo` обновляет или создает `ConfigDumpInfo.xml` для загруженной конфигурации.
- `-Extension <имя расширения>` возвращает `0` при успехе и `1`, если расширение не найдено или операция завершилась ошибкой.
