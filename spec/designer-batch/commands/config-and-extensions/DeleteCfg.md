# DeleteCfg

## Синтаксис

```text
/DeleteCfg [-Extension <имя расширения>] [-AllExtensions]
```

## Описание

— удаление расширений конфигурации. Если ни один параметр не задан, будет выдана ошибка. Доступны параметры:

### Параметры

- **Extension <Имя расширения>** — удаление расширений с указанным именем. Если расширение не существует или обнаружения иных ошибок,
код возврата — 1;
- **AllExtensions** — удаление всех расширений конфигурации.
