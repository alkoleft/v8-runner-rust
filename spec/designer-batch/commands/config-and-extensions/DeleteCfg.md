# DeleteCfg

## Назначение

Удаление расширений конфигурации. Если ни один параметр не задан, будет выдана ошибка. Доступны параметры:.

## Синтаксис

```text
/DeleteCfg [-Extension <имя расширения>] [-AllExtensions]
```

## Параметры

- `-Extension <Имя расширения>` — удаление расширений с указанным именем.
- `-AllExtensions` — удаление всех расширений конфигурации.

## Примечания

- Если ни один параметр не задан, будет выдана ошибка.
- `-Extension <Имя расширения>`: если расширение не существует или обнаружены иные ошибки, код возврата — 1;
