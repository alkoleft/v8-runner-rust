# CheckCanApplyConfigurationExtensions

## Назначение

Проверяет, можно ли применить расширения конфигурации.

## Синтаксис

```text
/CheckCanApplyConfigurationExtensions [-Extension <Имя расширения>] [-AllZones] [-Z <значения разделителей>]
```

## Параметры

- `-Extension <имя расширения>` — проверяет указанное расширение с учетом уже применяемых расширений.
- `-AllZones` — проверяет расширения во всех областях данных.
- `-Z <значения разделителей>` — запускает проверку для указанных разделителей.

## Связи

- Если `-Z <значения разделителей>` не указан, проверяются расширения вне разделенных данных.
- `-AllZones` нельзя использовать вместе с `-Extension <имя расширения>` или `-Z <значения разделителей>`.

## Примечания

- Если расширение из `-Extension <имя расширения>` не найдено, ошибка пишется в файл сообщений, заданный через `/Out`.
