# DumpConfigFiles

## Назначение

Выгружает отдельные свойства метаданных конфигурации: модули, макеты, картинки, права и справку.

## Синтаксис

```text
/DumpConfigFiles <каталог выгрузки> [-Module] [-Template]
[-Help] [-AllWritable] [-Picture] [-Right][-Extension <имя расширения>]
```

## Параметры

- `<каталог выгрузки>` — каталог для файлов свойств.
- `-Module` — выгружает модули.
- `-Template` — выгружает макеты.
- `-Help` — выгружает справочную информацию.
- `-AllWritable` — выгружает только свойства объектов, доступных для записи.
- `-Picture` — выгружает общие картинки.
- `-Right` — выгружает права.
- `-Extension <имя расширения>` — выгружает свойства указанного расширения.

## Примечания

- `-Extension <имя расширения>` возвращает `0` при успехе и `1`, если расширение не найдено или операция завершилась ошибкой.
