# LoadCfg

## Назначение

Загружает конфигурацию или расширение из `cf`-файла.

## Синтаксис

```text
/LoadCfg <имя cf-файла> [-Extension <имя расширения>]
```

## Параметры

- `<имя cf-файла>` — файл конфигурации.
- `-Extension <имя расширения>` — загружает файл в указанное расширение.

## Связи

- Если `-Extension <имя расширения>` подключено к хранилищу, загрузка невозможна.
- Если `-Extension <имя расширения>` не найдено, оно будет создано при загрузке.

## Примечания

- Если информационная база отсутствует в указанном каталоге, платформа создаст новую базу.
- `-Extension <имя расширения>` возвращает `0` при успехе и `1`, если расширение не найдено или операция завершилась ошибкой.
- Все каталоги в полном пути к `cf`-файлу должны существовать заранее.
