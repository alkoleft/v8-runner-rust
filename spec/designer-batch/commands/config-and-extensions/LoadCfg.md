# LoadCfg

## Синтаксис

```text
/LoadCfg <имя cf-файла> [-Extension <имя расширения>]
```

## Описание

— загрузка конфигурации из файла. Если файл информационной базы отсутствует в указанном каталоге, будет создана новая информационная база. Доступные параметры:

### Параметры

- **Extension <Имя расширения>** — обработка расширения с указанным именем. Если расширение успешно обработано возвращает код возврата 0, в противном случае (если расширение с указанным именем не существует или в процессе работы произошли ошибки) — 1. Если расширение не найдено, оно будет создано при загрузке конфигурации из файла.

**Важно!** Если указанное расширение подключено к хранилищу, загрузка конфигурации невозможна.

**Внимание!** Если в параметре командной строки требуется ввести имя файла, то следует учитывать, что при указании имени файла с полным путем все каталоги, входящие в состав пути, должны существовать.
