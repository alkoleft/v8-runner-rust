# MobileAppUpdatePublication

## Назначение

Обновляет публикацию мобильного приложения, если она была ранее создана, в противном случае будет выдана ошибка. Возможно предварительное обновление конфигурации базы данных.

## Синтаксис

```text
/MobileAppUpdatePublication
```

## Примечания

- — обновляет публикацию мобильного приложения, если она была ранее создана, в противном случае будет выдана ошибка.
