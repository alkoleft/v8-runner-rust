# DumpExternalDataProcessorOrReportToFiles

## Назначение

Выгружает внешнюю обработку или отчет в XML-файлы.

## Синтаксис

```text
/DumpExternalDataProcessorOrReportToFiles <путь к корневому файлу выгрузки>
<путь к файлу внешней обработки или отчета> [-Format Plain|Hierarchical]
```

## Параметры

- `<путь к корневому файлу выгрузки>` — корневой XML-файл выгрузки.
- `<путь к файлу внешней обработки или отчета>` — путь к `.epf` или `.erf`.
- `-Format Plain|Hierarchical` — формат выгрузки.

## Связи

- По умолчанию `-Format` использует `Hierarchical`.
- Дополнительно можно передать подключение к информационной базе, например через `/IBName`, чтобы разрешать внешние ссылки.

## Примечания

- `Hierarchical` создает структуру каталогов по объектам конфигурации.
- `Plain` выгружает все файлы в один каталог.
