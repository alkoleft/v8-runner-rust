# EraseData

## Назначение

Удаления данных информационной базы. С помощью ключа `/Z` задается область, в которой будут удалены данные. Удаление данных доступно пользователю с правом "Администрирование".

## Синтаксис

```text
/EraseData [/Z[<разделители>]]
```

## Связи

- Если в сеансе не используется ни один разделитель, то следующие настройки информационной базы заполняются значениями по умолчанию: - Время ожидания блокировки данных;
