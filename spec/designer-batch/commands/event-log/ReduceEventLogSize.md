# ReduceEventLogSize

## Назначение

Обрезает журнал регистрации по указанной дате.

## Синтаксис

```text
/ReduceEventLogSize <Date> [-saveAs <имя файла>] [-KeepSplitting]
```

## Параметры

- `<Date>` — новая граница журнала в формате `ГГГГ-ММ-ДД`.
- `-saveAs <имя файла>` — сохраняет копию выгружаемых записей.
- `-KeepSplitting` — сохраняет деление журнала на файлы по периодам.

## Примечания

- Все каталоги в полном пути к файлу `-saveAs` должны существовать заранее.
